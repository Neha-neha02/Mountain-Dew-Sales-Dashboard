# Mountain Dew Sales Analysis Dashboard

## Project Overview
This project presents a sales analysis dashboard created using Tableau. 
The dashboard helps visualize sales data and identify trends in revenue and regional performance.

## Dataset
The dataset used in this project contains sales information such as revenue, region, and sales channel.

## Tools Used
- Tableau
- Excel / CSV Dataset
- Data Visualization

## Dashboard Preview
The dashboard provides insights into:
- Month-wise revenue trends
- Region-wise sales performance
- Brand-wise sales comparison
